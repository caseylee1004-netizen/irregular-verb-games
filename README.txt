불규칙동사 붕어빵 게임 — 기존 GitHub 저장소 추가용

업로드할 파일
- index.html : 규칙변화동사와 불규칙동사를 함께 보여 주는 새 메인 화면
- irregular-verbs.html : 불규칙동사 게임 1~5 선택 화면
- irregular-game1.html : 1–20 (awake–dig)
- irregular-game2.html : 21–40 (do–have)
- irregular-game3.html : 41–60 (hear–pay)
- irregular-game4.html : 61–80 (put–sleep)
- irregular-game5.html : 81–100 (slide–write)

사용 방법
1. ZIP 압축을 풉니다.
2. 7개의 HTML 파일을 기존 GitHub 저장소의 regular-verbs.html과 같은 위치에 업로드합니다.
3. 기존 index.html은 새 index.html로 교체합니다.
4. 기존 regular-verbs.html 및 규칙동사 game1~5 파일은 삭제하지 않습니다.

음성
- 브라우저 Web Speech API의 en-US 미국 영어 음성을 우선 사용합니다.
- Chrome 또는 Edge 권장.
- read는 read /red/ /red/, tear는 /ter/ 발음이 나도록 음성용 값을 별도로 보완했습니다.
